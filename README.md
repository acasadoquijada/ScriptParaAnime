#ScriptAnime

##Uso del script
Para usar el script basta con abrir una terminal y lanzar el script junto con el anime y capitulo que queremos ver

Ej:
./animeflv.sh bleach 12

Si el anime tiene un nombre que se escribe separado como Shingeki no Kyojin, el script se usaria de la siguiente manera:

./animeflv.sh shingeki-no-kyojin 12

##Caracteristicas

* Permite ver de forma rapida y sencilla anime en una de las mejores paginas de anime en español

* Dispone de un registro de actividad para tener un control de los animes y capitulos vistos

* GUI

* Versiones estables en bash y python, esta última aún sin GUI

* Versión actual: 0.2
